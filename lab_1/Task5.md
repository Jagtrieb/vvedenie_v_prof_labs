# Выполнение задания 5

В задании просят попробовать скомпилировать из файла с разрешением .txt и из файла без разрешения. 

## Подготовка
Для выполнения задания нам потребуется терминал, компилятор C _(в примере будет использован компилятор **gcc**)_

## Выполнение задания 
Принцип выполнения задания для обоих файлов будет отличаться лишь названием файла с кодом, который вы хотите отдать компилятору. Действия один-в-один.  
**1.** Скомпилируйте код из файла с помощью следующей команды:  
`gcc -o {Название_исполнямеого файла} -x c {Название_файла_с_кодом}` 

**2.** Проверьте скомпилированный исполняемый файл, запустив его.

**Команды для терминала:**  
**MacOS/Linux**:  
`./{Название исполняемого файла}`

**Windows**:  
`{Название программы}.exe`
